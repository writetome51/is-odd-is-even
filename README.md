# isOdd(integer): boolean
# isEven(integer): boolean

## Examples
```
isOdd(2); // --> false

isEven(2); // --> true

isEven(2.2); // --> 'Error: input must be integer'

isEven('2'); // --> 'Error: input must be finite number of type "number"'
```

## Installation
`npm i  @writetome51/is-odd-is-even`

## Loading
